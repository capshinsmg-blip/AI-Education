## 슬라이드 1 — Claude Code란?

**본문:**
- 파일을 직접 읽고 고치고 실행까지 하는 코딩 에이전트
- 섹션 1 "손발 달린 비서"의 실물

**발표노트:** 섹션 1 슬라이드 5(에이전트)를 다시 띄워서 연결해도 좋음.

---

## 슬라이드 2 — git 개념도

**본문:**
```
내 PC (로컬) ←→ GitHub (원격) ←→ 다른 PC (로컬)
```
- 온라인 창고에 올려두면 어디서든 이어서 작업 가능

**발표노트:** "오늘 우리가 실제로 만든 저장소로 바로 해봅니다."

---

## 슬라이드 3 — 핵심 4동사

**본문:**
- clone: 처음 복제
- pull: 최신 받기
- commit: 변경 기록
- push: 온라인에 올리기

**발표노트:** 4개만 기억하면 된다고 안심시키기.

---

## 슬라이드 4 — 실습 명령 (1): clone

**본문:**
```bash
git clone https://github.com/capshinsmg-blip/AI-Education.git
```

**발표노트:** 참가자 본인 PC에서 실제로 실행해보게 함.

---

## 슬라이드 5 — 실습 명령 (2): commit & push

**본문:**
```bash
git add -A
git commit -m "update: 실습 수정"
git push origin main
```

**발표노트:** 파일 하나 살짝 수정 후 push까지 직접 해보게 함.
