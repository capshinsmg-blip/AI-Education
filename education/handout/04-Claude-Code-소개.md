# 섹션 4: Claude Code 소개 (30분, 실습)

## Claude Code란?

터미널(또는 에디터) 안에서 AI가 직접 파일을 읽고, 고치고, 명령까지 실행하는 **코딩 에이전트**다.
섹션 1에서 배운 "손발 달린 비서" 개념의 실물이 바로 이것이다.

---

## 온라인 git으로 다른 로컬 이동작업 (핵심)

### 개념도

```
내 PC (로컬)  ←→  GitHub (원격 저장소)  ←→  다른 PC (로컬)
```

작업 파일을 내 컴퓨터에만 두지 않고 GitHub라는 "온라인 창고"에 올려두면,
집 PC에서 하던 작업을 회사 PC에서 이어서 할 수 있고, 여러 사람이 같은 작업을 나눠 할 수도 있다.

### 오늘 실제로 만든 저장소로 실습

오늘 교육 자료 자체가 실제 예시다: `capshinsmg-blip/AI-Education` 저장소가 이미 만들어져 있다.

### 실습 절차

```bash
# 1. 다른 폴더(또는 다른 PC)에서 저장소 내려받기
git clone https://github.com/capshinsmg-blip/AI-Education.git

# 2. 파일을 수정한 뒤 온라인 저장소에 올리기
git add -A
git commit -m "update: 실습 수정"
git push origin main

# 3. 원래 PC에서 최신 내용 받아오기
git pull origin main
```

### 핵심 4동사

| 명령 | 뜻 |
|---|---|
| **clone** | 온라인 저장소를 처음 내 컴퓨터로 복제하기 |
| **pull** | 온라인 저장소의 최신 내용을 받아오기 |
| **commit** | 내가 바꾼 내용을 "기록"으로 남기기 |
| **push** | 내가 기록한 변경 내용을 온라인 저장소에 올리기 |

이 4개만 알면 "어디서든 이어서 작업하기"가 가능하다.

## 스킬 탐색법 및 다운로드 방법

### 스킬이란?

Claude Code에 특정 작업 능력을 더해주는 확장 기능. 문서 작성, 발표자료 제작, 데이터 분석 등
자주 하는 작업을 매번 처음부터 설명하지 않아도 되게 해준다.

### 탐색법

스킬/플러그인 목록에서 이름과 설명으로 원하는 기능을 찾는다.

- 문서 작성이 필요하면 → `docx` 관련 스킬
- 발표자료가 필요하면 → `pptx` 관련 스킬
- 표·데이터 작업이 필요하면 → `xlsx` 관련 스킬

### 설치·사용 방법

1. 마켓플레이스(플러그인 목록)에서 필요한 스킬을 찾아 설치
2. 설치 후 `/스킬이름` 형태로 호출해서 사용
3. 오늘 이 교육 자료도 `pptx` 스킬로 최종 슬라이드를 만들 예정 (섹션 마지막에 실물 확인)

### 실습

스킬 하나를 직접 찾아 설치하고, 간단히 호출해서 동작을 확인해본다.

> 참고: 화면 조작이 필요한 대화형 설정 메뉴는 실제 터미널에서 Claude Code를 실행할 때 사용할 수 있다.
