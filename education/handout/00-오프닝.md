# 오프닝 (10분)

## 도입 질문

> "오늘 아침에 AI를 몇 번 썼을까요?"

- 스마트폰 카메라 보정, 내비게이션 경로 추천, 유튜브 추천 영상, 메일 스팸 필터...
- 이미 우리는 하루에도 수십 번 AI를 쓰고 있다. 오늘은 "이미 쓰고 있는 AI"를 "의도적으로, 더 잘" 쓰는 법을 배운다.

## 오늘 배울 4가지 (한눈에 보기)

1. **AI의 종류와 의미** — 생성형 AI, LLM, 코딩 AI, 에이전트가 뭐가 다른지
2. **유용한 사이트 소개** — 폰트, 이미지, 레퍼런스를 공짜로 구하는 곳
3. **AI 프롬프트 입력법** — 오늘의 핵심. 원하는 결과를 얻는 질문법
4. **Claude Code 소개** — AI가 직접 파일을 고치고 작업해주는 도구 실습

## 이 교육의 약속

- **용어를 몰라도 됩니다.** 모든 개념은 비유로 설명하고, 어려운 말은 바로 풀어드립니다.
- **직접 해봅니다.** 특히 3, 4번은 실습 중심입니다. 노트북을 켜두세요.
- **정답은 없습니다.** 프롬프트도, 도구 선택도 "더 나은 방법"만 있을 뿐입니다. 오늘은 그 감을 잡는 시간입니다.
